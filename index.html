About

The Game of Life, also known as the Zero player Game is a game created by Cambridge Mathemetician John Conway in 1970. This game consists of a grid of cells which can be alive or dead. A pattern of units is given as input to the game. The game state changes and evolves by itself into a series of patterns.

The game evolves on the basis of two rules:

  • If two or three alive units surround an alive unit, it will remain alive. Otherwise it will die.
  • If three units surround a dead unit, it will become alive.
